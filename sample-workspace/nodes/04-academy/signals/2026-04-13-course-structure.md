## Summary

Academy splits into beginner + advanced tracks with a premium tier on top.

## Key points

- **Beginner**: 12 weeks, 3 modules, weekly live session.
- **Advanced**: 8 hands-on labs covering ingest → classify → embed →
  retrieve → curate.
- **Premium** ($10K): personal onboarding for enterprise.
- Co-teach split: Alice on beginner, Bob on advanced, Dan runs sales.

## Curriculum outline

### Beginner (12 weeks)

| Week | Topic                             |
|------|-----------------------------------|
| 1    | Signal Theory intro               |
| 2    | Three tiers: sources → derivatives → wiki |
| 3    | Classification `S=(M,G,T,F,W)`    |
| 4    | Entity extraction + routing       |
| 5    | Hierarchical chunking             |
| 6    | Multi-modal embedding             |
| 7    | Hybrid retrieval                  |
| 8    | Wiki curation + citations         |
| 9    | ACLs + principal-aware search     |
| 10   | Compliance workflows              |
| 11   | Connector integrations            |
| 12   | Capstone project                  |

### Advanced (8 labs)

1. Custom architecture registration
2. Processor behaviour implementation
3. Hybrid retrieval tuning
4. Graph boost + entity co-occurrence
5. Wiki curator integration
6. Audience-aware retrieval
7. Compliance automation
8. End-to-end production deploy

## Success criteria

Beginner cohort: 80% completion + NPS ≥ 50.
Advanced cohort: every student ships a working custom architecture.
